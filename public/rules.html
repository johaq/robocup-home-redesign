Existing teams
Want to improve the rules?

The rulebook evolves every year based on community feedback. If you have a suggestion for a new test, a clarification, or a change to existing rules, we welcome your input via pull request.

Open a pull request ↗ Browse the repository ↗
Changes are reviewed by the technical committee. The earlier in the year you propose, the better the chance of inclusion in the next release.
